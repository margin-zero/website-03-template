Website template based on BOOTSTRAP's Carousel

This template provides responsive structure for web page. As you already know - structure of this template is based on Carousel component from BOOTSTRAP framework. That means that every "page" of this template is single Carousel's slide. You can change "pages"/slides of the site using top navigation.

From technical point of view this template is One Page Website template so you put ALL your content into ONE html file.

If you need to add or remove "pages" from this template you need to do two things: first is to add or remove link in you navigation, and the second is to add / remove content section (div with .item class). After that be sure to check if data-slide-to attriutes in your navigation links have proper values - these numbers say which "slide" will be displayed when you use particular link (which div with .item class will be shown). Keep it in mind that "slides" are numbered from "0".

I have removed all unnecessary styles from CSS file of this template to give you control over your website's final look - left only basic document and menu styles and styles for thumbnails / preview elements (click the "FIRST LINK" in menu).

Building your website with this template should be easy but if you need you can take a look at the example website made using this template (or even use this example for your needs). If you don't have it yet, download it from my GitHub.

And finally... This template uses third party components (BOOTSTRAP, JQUERY) so respect proper licenses, images used in this template and example website are from UnSplash.com and are distributed under CC0 licence.
